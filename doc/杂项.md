```
# 1. 初始化本地仓库
git init

# 2. 将所有文件添加到暂存区
git add .

# 3. 将文件提交到本地仓库
git commit -m "Initial commit"

# 4. 添加远程仓库地址（替换成你复制的地址）
git remote add origin https://github.com/sumyunq/QT_OpenProject.git

# 5. 将本地分支重命名为 main（GitHub 默认分支）
git branch -M main

# 6. 首次推送到远程仓库（-u 参数用于建立关联，之后可直接用 git push）
git push -u origin main

https://github.com/sumyunq/QT_OpenProject.git
```
